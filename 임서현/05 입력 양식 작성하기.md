# 1. 폼 삽입하기

---

폼이란 사용자가 웹 사이트로 정보를 보낼 수 있는 모든 요소를 의미한다.

### 1.1 폼 만들기 `<form>`

- 속성
    - **`method`** : 입력한 내용을 서버로 어떻게 넘길지 지정
        - **get**
            - 256~4096byte 제한
            - 입력 내용 주소에 나타남
        - **post**
            - 길이 제한 없음
            - 입력 내용 표시 안 됨
    - **`name`** : 자바스크립트로 폼 제어를 위한 폼 이름 지정
    - **`action`** : 처리할 서버 지정
    - **`target`** : action 속성에서 지정한 스크립트 파일을 현재 창이 아닌 다른 위치에서 오픈
    - autocomplete : 이전 입력 내용 기반 자동 완성 on(default)/off

### 1.2 폼 그룹으로 묶기 `<fieldset>`, `<legend>`

- **<fieldset>** : 하나의 폼 안에서 구역을 나누어 표시
- **<legend>** : fieldset으로 묶은 그룹에 제목 할당

```jsx
<form>
    <fieldset>
        <legend>상품 선택</legend>
    </fieldset>
</form>
```

### 1.3 레이블 붙이기 <label>

<label>은 폼 요소에 레이블을 붙일 때 사용한다. 레이블과 폼 요소와 연결 방법은 총 두 가지가 있다.

- 방법 1

```jsx
<form>
    <label>아이디 <input type ="text"></label>
</form>
```

- 방법 2

```jsx

<form>
    <label for="id명">아이디</label>
    <input id="id명" >
</form>
```

# 2. input 태그

---

input 태그는 사용자가 입력한 정보를 받을 때 사용한다. 입력 형태를 지정하기 위해 type 속성을 사용한다.

| 타입 | 설명 |  |
| --- | --- | --- |
| `text` | 기본 텍스트 입력 | **text, password 속성**
- size : 필드 길이 지정
- value : 필드 부분에 보여주는 내용
- maxlength : 입력 최대 문자 수 지정 |
| `password` | 비밀번호 입력(문자 숨김) |  |
| `email` | 이메일 형식 입력 검증 |  |
| `number` | 숫자 입력 (스핀박스) | **number, range 속성**
- min : 최소 값 지정 (default = 0)
- max : 최대 값 지정 (default = 100)
-step : 숫자 간격 지정 (default =1)
-value : 필드에 표시할 초기값 |
| `range` | 숫자 범위 입력(슬라이더) |  |
| `tel` | 전화번호 입력 |  |
| `url` | URL 형식 입력 |  |
| `search` | 검색창 형태 입력 |  |
| `date` | 날짜 선택 |  |
| `time` | 시간 선택 |  |
| `datetime-local` | 날짜 + 시간 선택(지역 기반) | **date, time, datetime-local 속성**
- min : 최소 범위 지정
- max : 최대 범위 지정
- step : 스핀 박스 증감 사이즈
- value : 기본 표시 값 |
| `month` | 월 선택 |  |
| `week` | 주 선택 | 1월 첫째 주 기준 |
| `color` | 색상 선택기 표시 |  |
| `checkbox` | 체크박스 (중복 선택) | **라디오, 체그박스 속성**
- value : 서버에 넘겨줄 값 지정 (영문/숫자)
- checked : 기본 선택 값 설정
- name : 이름 지정 |
| `radio` | 라디오 버튼 (단일 선택) |  |
| `file` | 파일 업로드 |  |
| `hidden` | 화면에 표시되지 않음 |  |
| `button` | 일반 버튼 | **버튼 속성**
- value : 버튼에 표시될 내용 |
| `submit` | 폼 제출 버튼 |  |
| `reset` | 폼 초기화 버튼 |  |
| `image` | 이미지 버튼(폼 제출) |  |
- 라디오 버튼과 체크 박스는 하나의 그룹으로 동작하므로 name 속성에서 같은 이름을 사용해 묶는다.

## 2.1 주요 속성

- autofocus : 입력 커서 자동 할당
- placeholder : 입력란 힌트
- readonly : 읽기 전용 변경
- required : 필수 입력 지정

# 3. 그 밖의 태그

---

### 3.1 `<textarea>`

여러 줄을 입력하는 텍스트 영역

- 속성
    - cols : 텍스트 영역 가로 너비를 문자 단위로 지정
    - rows : 텍스트 영역 세로 길이를 줄 단위로 지정

---

### 3.2 드롭다운 목록 `<select>`, `<option>`

여러 옵션 중에서 선택하게 할 때 사용한다.

- **`<select>`**: 드롭 다운 목록의 시작과 끝을 표시
    - 속성
        - **size** : 화면에 표시할 드롭다운 항목 개수 지정
        - **multiple** : 드롭다운 목록에서 둘 이상의 항목 선택시 사용
- **`<option>`** : select 내 들어갈 value  값 지정
    - 속성
        - **vlaue** : 선택시 서버로 넘겨줄 값 지정
        - s**elected** : 기본 선택 값 지정
