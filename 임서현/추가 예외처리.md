# 1. 예외처리

---

## 1.1 기본 예외 처리

- .addEventListener() 이벤트 처리를 이용한 예외 발생
- window.event 이벤트 처리를 이용한 예외 발생

→ 즉, 이벤트 처리를 이용한 예외 발생

## 1.2 고급 예외 처리

`try` - `catch` - `finally`를 이용한 예외처리

*일반적으로 기본 예외 처리가 약간 더 빠르지만 차이가 거의 없을 뿐더러 뭐가 더 좋다고 판단 할 수 없기에 편한걸 사용하면 된다.*

## 1.3 예외 객체

예외 객체 속성은 브라우저마다 다르다. 아래는 공통 속성이다.

| 속성 이름 | 설명 |
| --- | --- |
| `message` | 예외 메시지 |
| `description` | 예외 설명 |
| `name` | 예외 이름 |

## 1.4 예외 강제 발생

예외를 강제로 발생시킬 땐 throw를 사용한다.

```jsx
<script>
	throw 'CUSTOM EXCEPTION OCCUR'
</script>
```
