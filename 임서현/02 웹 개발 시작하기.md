# 1. 웹 개발의 기본

---

## 1.1 프론트엔드 기술

- **HTML** : 웹 브라우저 창에 문서 내용을 보여준다.
- **CSS** :  HTML로 만든 내용을 꾸민다.
- Ja**vaScript** : 스크롤, 팝업창 보여주기 등 사용자 동작에 반응한다.

**JavaScript 의 라이브러리와 프레임워크**

- 라이브러리
    - **Jquery** : DOM을 활용해 웹 요소를 쉽게 사용하기 위한 라이브러리
    - **D3.js** : 실시간으로 변하는 정보를 시각적으로 표시하기 위한 라이브러리
    - **부트스트랩** : 웹 사이트 디자인을 쉽게 만들어주는 라이브러리
- 프레임워크
    - **리액트** : SPA(Single page application) 나 복잡한 사이트를 개발할 때 많이 사용하는 프레임워크
    - **앵귤러** : 웹 애플리케이션을 제작하는 프레임워크
    
    (프로젝트 성격에 따라 리액트, 앵귤러 중 선택해 사용한다.)
    
    - **뷰** : 사용자 인터페이스를 만드는 프레임워크, 화면에 보이는 부분에만 집중한다.

## 1.2 웹 개발자 도구

- **단축키** : `Ctrl` + `Shift` + `i`
- 웹 개발자 도구\요소 : CSS 속성 표시
