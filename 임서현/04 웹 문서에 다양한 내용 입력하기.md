# 1. 텍스트

---

### 1.1 제목 `<hn>`

```jsx
<h1>레드향</h1>
<h2>레드향</h2>
<h3>레드향</h3>
...
<h6>레드향</h6>
```

`<hn>` 은 제목을 나타내는 태그로 <h1> → <h6> 순으로 작아진다.

<aside>
💡

**추가**

- <h1>은 한 페이지에 1 개만 사용
- <h1> → <h6> 순서대로 사용
</aside>

### 1.2 단락 `<p>`

<p>는 텍스트를 단락으로 묶어 사용할 때 사용한다. 

브라우저 창에 맞춰 줄이 바뀐다.

### 1.3 줄변경 `<br>`

<br>은 원하는 위치에 줄을 바꿔 표시할 때 사용한다.

### 1.4 가로줄 <hr>

<hr>은 가로줄을 표시할 때 사용한다. 최근에는 여러 내용을 나열하는 경우 구분을 위해 주로 사용한다.

### 1.5 인용 `<blockquote>`

![image.png](attachment:19ffbbbe-d579-4ae2-afc9-dcc74b5dfe0a:image.png)

<blockquote>는 인용문을 표시할 때 사용한다. 사용시 다른 텍스트보다 약간 들여 쓴다.

### 1.6 텍스트 강조 `<strong>`, `<b>`

<strong>와 <b>는 텍스트를 강조하거나 굵게 표시할 때 사용한다.

육안으로는 구분하기 어렵지만 화면 낭독기에서 <strong>을 더 강조해서 읽어주는 차이가 있다.

- 화면 낭독기 : 시각 장애인을 위해 화면 정보를 음성으로 들려주는 프로그램

### 1.7 기울기 `<em>`, `<i>`, `<cite>`

<em>, <i>, <cite> 는 이탤릭체를 표시할 때 사용하며 각각 쓰임에 구분이 존재한다.

- **<em>** : 이탤릭체로 강조할 텍스트
- **<i>** : 이탤릭체로 표시할 텍스트
- **<cite>** : 저작물 제목

# 2. 목록 만들기

---

### 2.1 순서 목록 `<ol>`, `<li>`

<ol>과 <li>는 순서 목록을 만드는 태그로 앞 뒤에 <ol></ol> 을 두고 그 사이에 <li>와 <li> 를 삽입한다.

앞에 `1.` - `2.` - `3.` … 와 같은 숫자가 붙는다.

```jsx
<ol>
	<li>항목1</li>
	<li>항목2</li>
</ol>
```

### 2.2 순서 없는 목록 `<ul>`, `<li>`

<ul>과 <li>는 순서 없는 목록을 만드는 태그로 앞 뒤에 <ul></ul> 을 두고 그 사이에 <li>와 <li> 를 삽입한다.

앞에 `●` 와 같은 불릿이 붙는다.

```jsx
<ul>
	<li>항목1</li>
	<li>항목2</li>
</ul>
```

### 2.3 설명 목록 `<dl>`, `<dt>`, `<dd>`

설명 목록이란 이름 - 값 형태로 구성된 목록이다.

앞 뒤에 <dl></dl>을 두고 이름은 <dt></dt>로, 값은 <dd></dd>를 이용해 작성한다.

이때 <dd>는 여러개 사용할 수 있다.

# 3. 표만들기

---

### 3.1 표 `<table>`, `<caption>`

표를 만들 때는 앞 뒤로 <table></table>을 두고 이 사이에 표와 관련된 모든 내용을 넣는다.

<caption>은 표 제목으로 위쪽 중앙에 표시된다.

### 3.2 행 `<tr>`, 태그 `<th>`, 셀 `<td>`

<table> 태그 안에 행이 몇 개인지, 각 행에는 셀이 몇 개인지 지정해야 하기 때문에

하나의 셀을 만들기 위해서는 <tr>-<td>, <tr>-th>가 짝지어야 한다.

- `th` : 제목 행에 셀을 만들 때 사용.
굵은 글씨 + 가운데 정렬 된다.
- `td` : 제목 행이 아닌 셀을 만들 때 사용

### 3.3 표의 구조 `<thead>`, `<tbody>`, `<tfoot>`

<thead>, <tbody>, <tfoot>는 표의 구조를 지정할 때 사용한다.

- <thead> : 제목
- <tbody> : 본문
- <tfoot> : 요약 (합계나 요약 내용 등…)

### 3.4 행, 열 합치는 속성 `rowspan`, `colspan`

- rowspan : 행 합치기
- colspan : 열 합치기

```jsx
<td rowspan = "합칠 셀의 개수"> 내용 </td>
<td colspan = "합칠 셀의 개수"> 내용 </td>
```

### 3.5 열 묶기 `<col>`, `<colgroup>`

특정 열에 배경색을 넣거나 너비를 바꿀 때 사용한다.

- <col> :  열 1개 지정
- <colgroup> : <col> 태그를 2개 이상 묶어서 사용

```jsx
<colgroup>
	<col>
	<col style="width:150px;">
	<col style="background-color:#eee;">
</colgroup>
```

- 반드시 <caption> 태그 다음에 사용
- <colgroup> 태그 안에 표 전체 열 개수 만큼 <col> 생성
    - 위 코드의 경우 총 3개의 열이 있는 표로 1열 스타일X, 2열 너비 150px, 3열 배경색 #eee 지정을 의미한다.

# 4. 이미지 삽입하기

---

### 4.1 이미지 삽입 <img>

```jsx
<img src="이미지 파일 경로" alt="대체 텍스트">
```

- alt : 화면 낭독기나 인터넷 불안정 할 때 이미지를 대신해서 읽어 줄 텍스트 입력
- src : 이미지 경로(기준 : 웹 문서 파일 위치)

### 4.2 웹 요소에 설명 글 붙이기 <figure>, <figcaption>

- <figure> : 도표, 일러스트, 이미지, 소스 코드 등 독립된 콘텐츠를 나타낼 때 사용한다.
- <figcatpion> : <figure> 내부에 위치하며 콘텐츠에 태그를 사용해 설명을 붙일 때 사용한다.

### 4.3 이미지 삽입 srcset

```jsx
<img
  srcset="images/heropy_small.png 400w,
          images/heropy_medium.png 700w,
          images/heropy_large.png 1000w
          images/heropy_large.png 1x
          "
          
  src="images/heropy.png"
  alt="HEROPY" />
```

srcset은 브라우저에 제시할 이미지들과 그 이미지의 원본 크기를 지정한다.

- 속성
    - w : 파일 너비 지정 (not px)
    - x : 장치의 픽셀 비율 (클수록 고화질 이미지)
- 역할
    - PC/모바일 에 따라 표시할 이미지 지정
    - 모니터 해상도에 따른 이미지 파일 표시 등…

# 5. 오디오와 비디오 삽입하기

---

### 5.1 멀티미디어 파일 삽입 `<object>`, `<embed>`

- **<object>**는 오디오 파일뿐만 아니라 비디오, PDF 등 멀티미디어 파일을 삽입할 때 사용한다.
- **<embed>**는 오래된 브라우저에 멀티미디어 파일을 삽일할 때 사용한다.

### 5.2 오디오와 비디오 파일 삽입 `<audio>`, `<vido>`

```jsx
<audio src="오디오 파일 경로"></audio>
<vidio src="비디오 파일 경로"></vidio>
```

- 속성
    - **controls** : 플레이어 화면에 컨트롤 바 표시
    - **autoplay** : 오디오나 비디오 자동 실행
    - **loop** : 오디오나 비디오 반복 재생
    - **muted** : 오디오나 비디오 소리 제거
    - **preload** : 파일 로딩 방법 지정
        - auto(default), metadata, none
    - **width, height** : 너미와 높이 지정
    - **poster=”파일 이름”** : <video>에서 사용, 비디오 재생 전까지 표시할 이미지 요소 지정
    

*추가) 웹 문서에서 멀티미디어 파일을 사용 할 때 주의 사항*

- 사용자 동의 없이 재생 금지
- 자동 재생이 필요한 경우 소리 제거

# 6. 하이퍼링크 삽입하기

---

### 6.1 링크 삽입 <a>, href

```jsx
<a href="링크할 주소"> 텍스트 또는 이미지 </a>
```

이때, 텍스트를 사용하면 텍스트 링크, 이미지를 사용하면 이미지 링크가 된다.

- 속성
    - **target = “_blank”** : 새 탭에서 링크 오픈

### 6.2 앵커

앵커란 한 페이지 안에서 이동하는 링크이다.

```jsx
<태그 id="앵커명"> 내용 </태그>
<a href="#앵커명">링크</a>
```

- 이동하고 싶은 위치에 id 속성을 이용해 앵커 이름을 붙이고, `#`으로 앵커임을 표시한다.
