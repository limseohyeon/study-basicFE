# 1. HTML 구조

```html
<!DOCTYPE html>

<html lang="ko">
    <head>
        <meta charset ="UTF-8">
        <title> HTML 기본 문서 </title>
    </head>

    <body>
        <h1>웹 문서 만들기</h1>
        <hr>
        <p>HTML</p>
        <p>CSS</p>
        <p>자바 스크립트</p>
    </body>
</html>
```

## 1.1 구조 분석

### 1.1.1 `<!DOCTYPE html>`

현재 문서가 HTML로 작성한 웹 문서임을 의미

### 1.1.2 `<html>` ~ `</html>`

웹 문서의 시작과 끝 의미

### 1.1.3 `lang=”ko”`

검색시 해당 언어로 제한해 검색 한다. 또한, 음성 낭독기 등 언어 맞춤 기능 제공

### 1.1.4 `<head>` ~ `</head>`

스타일 시트 등 브라우저가 웹 문서를 해석하는 데 필요한 정보 입력.

그 중 <meta> 와 <title>부터 보도록 한다.

**`<meta>`** 

- 웹 브라우저에 문서 정보를 알림
- meta 사용 예시
    - <meta charset = “UTF-8”> : 한글을 표시하기 위해 UTF-8 인코딩 사용
    - <meta name =”keywords” content=”html구조”> :  웹 문서 키워드
    - <neta name = ”description” content =”html의 구조 알아보기”> : 웹 문서의 설명
    - <meta name=”author” content=”Kyunghee Ko”> : 웹 문서 소유자 or 제작자

`<title>` 

- 문서 제목, 엡 브라우저의 제목 표시줄에 표시

### 1.1.5 `body>` ~ `</body>`

실제 웹 브라우저 화면에 나타나는 내용

## 1.2 태그 작성 시 주의사항

- 태그는 소문자로 작성
- 여는 태그와 닫는 태그 정확하게 입력
- 태그는 여러 칸 있어도 한 칸으로 인식함, 적절한 들여쓰기 필요
- 일부 태그는 속성과 함께 사용함 *ex) <img src = “images/salad.jpg” alt=”레드향” width=”150”>*

(+) HTML에서는 영어 대소문자를 구별하지 않아도 된다.

# 2. 시멘틱 태그

---

특별한 기능은 없으나, 보는 것으로 그 역할을 알 수 있는 특정 태그를 의미한다.

![](https://velog.velcdn.com/images%2Fsyoung125%2Fpost%2F21d321a8-5e5b-4f48-9635-3a217a25799d%2Fimage.png)

## 2.1 시맨틱 태그 종류

### 2.1.1 `<header>`

사이트의 헤더를 의미한다. 주로 맨 위쪽, 왼쪽에 있으며 검색 창이나 사이트 메뉴를 삽입한다.

### 2.1.2 `<nav>`

웹 문서 안에서 다른 위치 또는 다른 웹 문서로 연결하는 링크를 만든다.

웹 문서 위치에 영향을 받지 않기 때문에 헤더나 푸터, 사이드바 안에 포함 할 수 도있다.

### 2.1.3 `<main>`

핵심 콘텐츠가 담긴다. 웹 문서마다 다르게 보여주는 내용으로 구성하며 한 번만 보여줄 수 있다.

### 2.1.4 `<article>`

블로그 포스터, 뉴스 사이트 기사처럼 독립된 웹 콘텐츠 항목으로 여러 개 사용 가능, <section>을 넣을 수 도 있다.

### 2.1.5 `<section>`

콘텐츠 영역을 나타낸다.

**<article>, <div>, <section>차이**

- <article> : 콘텐츠 자체를 의미함
- <div> : 콘텐츠의 영역을 나타냄
- <section> : 독립된 콘텐츠의 영역을 나타냄
- 

### 2.1.6 `<aside>`

사이드 바 영역을 나타낸다. 필수는 아니다.

### 2.1.7 `<fooer>`

웹 문서 맨 아래쪽 사이트 제작 정보, 저작권 정보, 연락처 등을 넣는다. 마찬가지로 <script>, <article> 등 다른 시맨틱 태그 사용 가능하다.

### 2.1.8 `<div>`

여러 코드를 묶어 문서 구조를 만든다.

`<div id=”header”>`, `<div class =”detail”>` 처럼 id나 class 속성을 사용해 스타일을 적용할 수 있다.
