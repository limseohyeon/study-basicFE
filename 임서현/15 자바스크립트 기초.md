# 1. 자바스크립트 기초

---

## 1. 1 <script></script>

- 태그 내부에 실행할 자바스크립트 코드를 작성
- 어디든 위치 할 수 있고, 여러 개 사용 가능
- 대/소문자 구분 필요
- 되도록 </body> 태그 직전에 사용 (웹 문서 내부 요소를 제어하는 경우가 많기 때문 )

## 1.2 외부 스크립트 파일로 연결해 사용하기

`<script>` 태그 없이 코드만 작성하며 `*.js` 파일로 저장하고,

HTML의 src 속성을 이용해 파일을 연결해 사용한다.

```jsx
<script scr="외부 스크립트 파일 경로"></script>
```

## 1.3 웹 브라우저 스크립트 해석 순서

1. `<!DOCTYPE html>`를 보고 `<html></html>` 태그 사이 내용을 HTML표준에 맞춰 읽는다.
2. `<head></head>`와 `<body></body>` 사이 태그 순서와 포함 관계 분석
3. `<style></style>` 분석
4. 웹 브라우저 내 자바스크립트 해석기가 `<script>`내부 코드 해석
5. *(2)* 와 *(3)* 에 따라 브라우저에 화면 표시
6. 이벤트 발생 시 자바 스크립트 실행 

## 1.4 간단한 입출력

- `alert(메시지)` : 알림창, 메시지만 표시
- `confirm(메시지)` : 확인창, 메시지 + 버튼 2개
- `prompt(메시지, [기본값])` : 텍스트 필드가 있는 작은 창
- `document.write()` : 괄호 안의 내용을 표시하는 명령문
    - 변수, 텍스트, html 태그 등 다양한 값 표시
    - `+` : 각 요소를 연결하는 연결 연산자
- `console.log()` : 콘솔 창에 표시

자바스크립트 컨벤션

1. 들여쓰기 필수
2. 세미콜론으로 문장 구분
3. 예악어나 연산자, 값 사이에는 공백 추가
4. 코드 설명 주석 첨가
5. 식별자
    1. 첫 글자는 `영문자`, `_`, `$` 로 시작
    2. 단어 사이의 공백X, `_` 또는`-` 로 연결 혹은 Camel
    3. 예약어는 식별자로 사용 불가
