# 1. 브라우저와 관련된 객체

---

![image.png](attachment:8ae2d653-8f86-47d0-ac60-4dd3803521d0:image.png)

브라우저는 HTML 코드를 한 줄씩 읽으면서 화면에 내용을 표시하고 관련된 객체를 window를 시작으로 위와 같은 순서로 만들어 낸다.

## **1.2 브라우저 관련 객체 종류**

### 1.2.1 window 객체

window 객체는 브라우저 창이 열릴 때 마다 하나씩 만들어지며 창 안의 요소 중 가장 최상위에 있기 때문에 어디서든 접근이 가능하다.

**속성**

| 속성 | 설명 |
| --- | --- |
| `document` | 현재 웹 페이지의 문서 객체(DOM)에 접근 |
| `location` | 주소(URL) 정보 접근 및 페이지 이동 제어 |
| `history` | 방문 기록(뒤로/앞으로 이동) 제어 |
| `navigator` | 브라우저/운영체제 정보 |
| `screen` | 화면(모니터) 정보 |
| `localStorage` | 브라우저 영구 저장소 |
| `sessionStorage` | 세션(브라우저 종료 시 삭제) 저장소 |
| `innerWidth` / `innerHeight` | 브라우저 창 내부의 넓이와 높이 |
| `outerWidth` / `outerHeight` | 브라우저 창 전체의 넓이와 높이 |

**메서드**

| 메서드 | 설명 | 옵션 |
| --- | --- | --- |
| `alert(message)` | 경고창 표시 |  |
| `confirm(message)` | 확인/취소 창 표시 (true/false 반환) |  |
| `prompt(message)` | 입력창 표시 (입력값 반환) |  |
| `open(url)` | 새로운 브라우저 창 또는 탭 열기 | • height : 새 윈도우의 높이
• width : 새 윈도우의 너비
• location : 주소 입력창의 유무
• menubar : 메뉴의 유무
• resizable : 화면 크기 조절 가능 여부
• ,status : 상태 표시줄의 유무
• toolbar : 상태 표시줄의 유무 |
| `close()` | 현재 창 닫기 |  |
| `setTimeout(function, time)` | 일정 시간 후 함수 1회 실행 |  |
| `setInterval(function, time)` | 일정 시간 간격으로 반복 실행 |  |
| `requestAnimationFrame(callback)` | 애니메이션 최적화 실행 |  |
| `scrollTo(x, y)` | 페이지 스크롤 이동 |  |
| `print()` | 인쇄 창 열기 |  |

### 1.2.2 location 객체

웹 문서마다 하나씩 있으며 <body> 태그를 만나면 만들어 진다. HTML 문서 정보가 담겨 있다.

**속성**

| 속성 | 설명 | 예시 |
| --- | --- | --- |
| `innerWidth` | 현재 브라우저 창의 내부 가로 길이 | `window.innerWidth` |
| `innerHeight` | 현재 브라우저 창의 내부 세로 길이 | `window.innerHeight` |
| `document` | 현재 페이지의 문서(DOM)에 접근 | `window.document.getElementById()` |
| `localStorage` | 로컬 스토리지(브라우저 데이터 저장) | `localStorage.setItem()` |
| `sessionStorage` | 세션 스토리지 저장 공간 | `sessionStorage.getItem()` |
| **`location`** | **현재 페이지 URL 정보 & 이동 기능 제공 객체** | `window.location.href` |

**메서드**

| 메서드 | 설명 | 예시 |
| --- | --- | --- |
| `alert()` | 알림창 출력 | `alert("Hello")` |
| `confirm()` | 확인/취소 창 출력 | `confirm("삭제할까요?")` |
| `prompt()` | 입력창 출력 | `prompt("이름을 입력하세요")` |
| `setTimeout()` | 지정 시간 후 함수 실행 | `setTimeout(fn, 2000)` |
| `setInterval()` | 일정 시간마다 반복 실행 | `setInterval(fn, 1000)` |
| `open()` | 새 창 열기 | `window.open("https://google.com")` |
| `scrollTo()` | 화면 스크롤 이동 | `window.scrollTo(0, 300)` |

### 1.2.3 naviator 객체

현재 사용하는 브라우저의 정보가 들어있다.

**속성**

| 속성 | 설명 | 예시 |
| --- | --- | --- |
| `userAgent` | 브라우저·OS 정보 문자열 | `navigator.userAgent` |
| `language` | 현재 브라우저 언어 | `navigator.language` → `"ko-KR"` |
| `onLine` | 온라인 여부 (네트워크 연결 상태) | `navigator.onLine` |
| `cookieEnabled` | 쿠키 사용 가능 여부 | `navigator.cookieEnabled` |
| `geolocation` | 위치 정보 객체 | `navigator.geolocation.getCurrentPosition()` |
| `clipboard` | 클립보드 복사/붙여넣기 API | `navigator.clipboard.writeText("hello")` |
| `mediaDevices` | 카메라·마이크 장치 접근 | `navigator.mediaDevices.getUserMedia()` |

**메서드**

| 메서드 | 설명 | 예시 |
| --- | --- | --- |
| `getBattery()` | 배터리 정보 조회(지원 브라우저만) | `navigator.getBattery().then(b => console.log(b.level))` |
| `vibrate()` | 기기 진동(지원 기기에서만) | `navigator.vibrate(200)` |

### 1.2.4 history 객체

현재 창에서 사용자의 방문 기록을 저장한다.

**속성**

| 이름 | 설명 | 예시 |
| --- | --- | --- |
| `length` | 방문 기록의 개수(스택 크기) | `history.length` |

**메서드**

| 이름 | 설명 | 예시 |
| --- | --- | --- |
| `back()` | 이전 페이지로 이동 (`뒤로가기`) | `history.back()` |
| `forward()` | 다음 페이지로 이동 (`앞으로가기`) | `history.forward()` |
| `go(n)` | n칸 이동 (`+` 앞으로, `-` 뒤로) | `history.go(-2)` → 2칸 뒤로 |
| `pushState(state, title, url)` | 페이지 이동 없이 URL만 변경 | `history.pushState({}, "", "/home")` |
| `replaceState(state, title, url)` | 현재 기록을 새로운 정보로 교체 | `history.replaceState({}, "", "/login")` |

### 1.2.5 screen 객체

현재 페이지의 URL 정보가 담겨 있다.

**속성**

| 이름 | 설명 | 예시 |
| --- | --- | --- |
| `width` | 화면(모니터) 전체 너비 | `screen.width` |
| `height` | 화면(모니터) 전체 높이 | `screen.height` |
| `availWidth` | 작업 표시줄 제외 실제 사용 가능한 너비 | `screen.availWidth` |
| `availHeight` | 작업 표시줄 제외 실제 사용 가능한 높이 | `screen.availHeight` |
| `colorDepth` | 색상 비트 수 (일반적으로 24 or 32bit) | `screen.colorDepth` |
| `pixelDepth` | 실제 픽셀 비트 수 | `screen.pixelDepth` |

### 1.2.6 document 객체

현재 사용하는 화면 정보를 다룬다.

속성

| 속성 이름 | 설명 | 예시 |
| --- | --- | --- |
| `body` | `<body>` 요소에 접근 | `document.body` |
| `head` | `<head>` 요소에 접근 | `document.head` |
| `title` | 문서 제목 (읽기/쓰기 가능) | `document.title = "Hello"` |
| `URL` | 현재 문서의 전체 URL | `document.URL` |
| `cookie` | 쿠키 읽기/쓰기 | `document.cookie = "name=Tom"` |
| `domain` | 문서의 도메인 이름 | `document.domain` |
| `lastModified` | 문서의 마지막 수정 날짜 | `document.lastModified` |
| `readyState` | 문서 로딩 상태 | `document.readyState` → `"complete"` |
| `forms` | 문서에 있는 `<form>` 요소 목록 | `document.forms` |
| `images` | 문서에 있는 `<img>` 요소 목록 | `document.images` |

**메서드**

| 메서드 이름 | 설명 | 예시 |
| --- | --- | --- |
| `getElementById(id)` | id로 요소 1개 선택 | `document.getElementById("box")` |
| `getElementsByClassName(class)` | class명으로 요소 여러 개 선택 | `document.getElementsByClassName("item")` |
| `getElementsByTagName(tag)` | 태그명으로 요소 여러 개 선택 | `document.getElementsByTagName("li")` |
| `querySelector(selector)` | CSS 선택자로 요소 1개 선택 | `document.querySelector("#box")` |
| `querySelectorAll(selector)` | CSS 선택자로 요소 여러 개 선택 | `document.querySelectorAll(".item")` |
| `createElement(tag)` | 새 HTML 요소 생성 | `document.createElement("div")` |
| `createTextNode(text)` | 텍스트 노드 생성 | `document.createTextNode("Hello")` |
| `appendChild(node)` | 노드를 자식으로 추가 | `parent.appendChild(child)` |
| `removeChild(node)` | 자식 노드 삭제 | `parent.removeChild(child)` |
| `write(text)` | 문서에 문자열 출력 (실전 거의 X) | `document.write("Hi")` |
| `addEventListener(event, handler)` | 문서에 이벤트 리스너 등록 | `document.addEventListener("click", fn)` |
